Password Element

The following method will add a password input element to the form utilizing the form_password() form helper function.

password (string name|id [, string label, string rules, string value, string|array attributes])

  • name|id: adds name and id attributes to the element (where id is optional)
  • label: adds a label element to the label position
  • rules: adds validation rules to the form validation
  • value: adds a default value to the element
  • attributes: adds other valid attributes to the element (as string or array)

Note: If the 'id' is not specified but needed for the label tag, the library will automatically create a unique id for you.

Usage Example

$this->form->password('pass', 'Your Password', 'trim|callback_valid_pwd|xss_clean|md5'); will produce <label for="iduq6d">Your Password</label><input type="password" name="pass" id="iduq6d" value="" /> and validate the trimmed input value with the callback function valid_pwd(), perform an XSS clean and encode the value with MD5.
