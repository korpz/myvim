Label Element

The following method will add a label to the form utilizing the form_label() form helper function.

label (string label, string for [, string|array attributes])

  • label: adds a label element to the form
  • for: contains the id of the element that the label is for
  • attributes: adds other valid attributes to the label

Note: You only need to use the label function if the label is not placed in the default label position next to its corresponding element.

Usage Example

$this->form
->label('Zip/', 'zip')
->label('City', 'city')
->text('zip|zip', '', 'max_length[5]')
->text('country|country') will produce <label for="zip">Zip/</label>
<label for="city">City</label>
