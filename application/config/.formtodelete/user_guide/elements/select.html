Select Element

The following method will add a select element to the form utilizing the form_dropdown() form helper function.

select (string name|id, array $options [, string label, string|array selected, string rules, string|array attributes])

  • name|id: adds name and id attributes to the element (where id is optional)
  • options: an array containing the select options
  • label: adds a label element to the label position
  • selected: an array or comma separated string with default values
  • rules: adds validation rules to the form validation
  • attributes: adds other valid attributes to the element (as string or array)

Note: This method will return the selected values as an array.

Usage Example

// first we need to populate the options array
$single = array(
    1 => 'One',
    2 => 'Two',
    3 => 'Three'
);

// then we're creating the select element
$this->form->select('single', $single, 'Single Select', 2); which will produce <label for="kshqi2">Single Select</label>
<select name="single" id="kshqi2">
   <option value="1">One</option>
   <option value="2" selected="selected">Two</option>
   <option value="3">Three</option>
</select>

Creating Multiple Selects With Groups

To create a multiple select simply add the attribute "multiple=multiple" or "multiple=TRUE" or use the method multiple().

Note: If more than one default value is provided the select element will automatically become a multiple select.


To add groups to the select element simply feed the 'options' parameter with a multidimensional array containing the optgroups. // first we need to populate the options array
$multiple = array(
    'Group X' = array(
       '1' => 'One',
       '2' => 'Two',
       '3' => 'Three'
    ),
    'Group Y' = array(
       '4' => 'Four',
       '5' => 'Five'
    )
);

// then we're creating the select element
$this->form->select('multiple|multi', $multiple, 'Multiple Select', 2,5, 'xss_clean', 'style=height: 150px'); which will produce <label for="multi">Multiple Select</label>
<select name="single" id="multi" multiple="multiple" style="height:150px">
<optgroup label="Group X">
   <option value="1">One</option>
   <option value="2" selected="selected">Two</option>
   <option value="3">Three</option>
</optgroup>
<optgroup label="Group Y">
   <option value="4">Four</option>
   <option value="5" selected="selected">Five</option>
</optgroup>
</select>
