Fieldset Element

The following method will add a fieldset element to the form utilizing the form_fieldset() form helper function.

fieldset ([string legend, string|array attributes])

  • legend: adds a legend tag to the element
  • attributes: adds other valid attributes to the element (as string or array)

Note: You don't need to worry about closing your fieldset. The library will automatically take care of it.

Usage Example

$this->form->fieldset('Personal Data', 'class=fset'); // note that both 'class=fset' or 'class="fset"' (value enclosed by double quotes) will work as an attribute parameter will produce <fieldset class="fset">
<legend>Personal Data</legend>
