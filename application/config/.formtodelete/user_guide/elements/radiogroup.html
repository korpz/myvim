Radiogroup Element

The following method will add multiple radiobutton elements to the form utilizing the form_radio() form helper function.

radiogroup (string name, array $radios [, string label, string|array checked, string rules, string|array attributes])

  • name: adds name attributes to the element (id's can be provided with $radios parameter)
  • radios: an array containing the radiobuttons
  • label: adds a label element to the label position
  • checked: an array or comma separated string with default values
  • rules: adds validation rules to the form validation
  • attributes: adds other valid attributes to the element (as string or array)

Note: This method will automatically add [] to the element name and therefore return the checked value as an array (or as string when returned via get_post() method).

Usage Example

// first we need to populate the options array
$radios[] = array('this', 'This');
$radios[] = array('or', 'Or');
$radios[] = array('that', 'That');

// then we're creating the radiogroup element
$this->form->radiogroup('selectone', $radios, 'Please choose:', '', 'required') which will produce <label>Please choose:</label>
