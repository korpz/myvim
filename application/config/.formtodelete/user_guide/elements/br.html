Break Element

The following method will add a line break to the form.

br ([bool clear])

  • clear: if set to TRUE 'style="clear:both"' will be added to the <br> tag

Usage Example

$this->form
->text('city|city', 'City', 'trim|xss_clean')
->br()
->text('zip|zip', 'Zip Code', 'trim|xss_clean'); will produce <label for="city">City</label><input type="text" name="city" id="city" value="" /><br />
