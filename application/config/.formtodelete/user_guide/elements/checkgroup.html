Checkgroup Element

The following method will add multiple checkbox elements to the form utilizing the form_checkbox() form helper function.

checkgroup (string name, array $checks [, string label, string|array checked, string rules, string|array attributes])

  • name: adds name attributes to the element (id's can be provided with $checks parameter)
  • checks: an array containing the checkboxes
  • label: adds a label element to the label position
  • checked: an array or comma separated string with default values
  • rules: adds validation rules to the form validation
  • attributes: adds other valid attributes to the element (as string or array)

Note: This method will automatically add [] to the element name and return the checked values as an array.

Usage Example

// first we need to populate the options array
$checks[] = array('spo|c1', 'Sports', 'style=font-style: italic');
$checks[] = array('lit|c2', 'Literature', 'style=font-style: italic');
$checks[] = array('foo|c3', 'Food', 'style=font-style: italic');

// then we're creating the checkgroup element
$this->form->checkgroup('interests', $checks, '', 'lit', 'required') which will produce <input type="checkbox" name="interests[]" id="c1" value="spo" /><label for="c1" style="font-style: italic">Sports</label>
<input type="checkbox" name="interests[]" id="c2" value="lit" checked="checked" /><label for="c2" style="font-style: italic">Literature</label>
<input type="checkbox" name="interests[]" id="c3" value="foo" /><label for="c3" style="font-style: italic">Food</label> and look like
