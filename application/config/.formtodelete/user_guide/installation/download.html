Upgrade instructions from previous versions to 1.0

Version 1.0 includes major updates. Please make sure that you backup your data before you apply changes to your controllers for this version of the library.


Check all your controllers and make adjustments for the following important changes:
  • $config['break_after'] has been replaced completely by the prefix/suffix templating system
  • nobr() method has been removed completely
  • $config['label_pos'] has been removed completely
  • $config['label_req_class'] has been removed completely
  • $config['label_req_flag'] has been removed completely
  • Label appearance and behaviour is now set via $config['defaults']
  • flush() method has been replaced by clear() method
  • Make sure that you update your config file
  • Make sure that you update your style sheet
