Set Error (Message)

This method allows you to set a custom error message for any form element. This is especially useful for radiogroups, checkgroups and checkbox elements without labels.

Note: You can use the {label} placeholder to include the element's label in your custom error message.

set_error (string message)

  • message: the error message

This method usage sets an error message for the last accessed element.

set_error (string name, string message)

  • name: the element name
  • message: the error message

This method usage sets an error message for any element of the form.

Usage Example

->checkbox('question', 'yes', 'I agree', FALSE, 'required')
->set_error('You must agree to the question!'); will change the default error messsage:
The question field is required.
into
You must agree to the question!
