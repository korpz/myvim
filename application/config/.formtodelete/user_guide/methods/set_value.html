Set Value

This method allows you to set a value for any form element.

set_value (string|integer value)

  • value: the element value

This method usage sets a value for the last accessed element.

set_value (string name, string|integer value)

  • name: the element name
  • value: the element value

This method usage sets a value for any element of the form.

Usage Example

$this->form
->text('zip');
->text('city');
->set_value('Munich') // for last accessed element
->set_value('zip', '80339') // for specific element will result in <input type="text" name="zip" value="80339" />
<input type="text" name="city" value="Munich" />

Note: This method can be useful for altering form elements from within a model.
