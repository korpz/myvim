Remove Attribute

The following method will allow you to remove an attribute from an element. This can be useful if you assign certain attributes to elements by default via the config file and need to remove those attributes for certain elements later.

rem_att (string attribute)

  • attribute: the attribute name

This method usage removes an attribute from the last accessed element.

rem_att (string element, string attribute)

  • element: the element name
  • attribute: the attribute name

This method usage removes an attribute from any element of the form.

Usage Example

// assuming that the attribute 'maxlength' is added to every element of the type 'text' by default

$this->form
->text('fname');
->text('mname');
->text('lname');
->rem_att('mname', 'maxlength') will result in <input type="text" name="fname" maxlength="255" />
<input type="text" name="mname" />
<input type="text" name="mname" maxlength="255" />
