Get Post

The following method will allow you to retrieve the validated $_POST data of the submitted form.

get_post (bool xss_clean)

  • xss_clean: if set to TRUE, all post variables will be xss cleaned

You can set the xss_clean value globally in the config file.

Note: Radiogroups will be returned as string whereas checkgroups and selects will always be returned as arrays.

Usage Example

The following example is based on the welcome.php controller of the installation package. 
$data['form'] = $this->form->get();

if ($this->form->valid)
{
    $post = $this->form->get_post();
   
    echo '<pre>'.print_r($post, TRUE).'</pre>';
}
