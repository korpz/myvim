Indent

This method will add an indentation (left margin) to all following elements. To reset the indentation simply supply the value of 0.

indent (integer width)

  • width: width of indentation in pixels, 0 for resetting indentation

Usage Example

$this->form

->text('name', 'Your name')

// indent the following element because it has no label
->indent(150)

// add an element without a preceding label
->checkbox('question', 'yes', 'I agree')

// reset indentation
->indent(0)

->text('another', 'Another Input')
