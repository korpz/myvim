Set Attribute

The following method will allow you to set an attribute for an element.

set_att (string attribute, string|integer value)

  • attribute: the attribute name
  • value: the attribute value

This method usage sets an attribute for the last accessed element.

set_att (string name, string attribute, string|integer value)

  • name: the element name
  • attribute: the attribute name
  • value: the attribute value

This method usage sets an attribute for any element of the form.

Usage Example

$this->form
->text('zip');
->text('city');
->set_att('readonly', 'readonly') // for last accessed element
->set_att('zip', 'maxlength', '5') // for specific element will result in <input type="text" name="zip" maxlength="5" />
<input type="text" name="city" readonly="readonly" />
