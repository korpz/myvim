Clear

The following method will allow you to use multiple forms in one controller. It will clear all data from the previous form.

clear ()

This method needs to be called in between the generation of two forms.

$this->load->library('form');

$this->form
->text('Name', 'name')
->submit()
->model('user', 'update')
->onsuccess('redirect', 'success');

$data['form1'] = $this->form->get();

$this->form->clear();

$this->form
->text('Name', 'name')
->submit()
->model('user', 'update')
->onsuccess('redirect', 'success');

$data['form2'] = $this->form->get();

Multiple Forms

Note: In order to use multiple forms in one controller each form must have unique submit names.

// form 1
->submit('Submit Form 1', 'submit_form1')

// form 2
->submit('Submit Form 2', 'submit_form2')
