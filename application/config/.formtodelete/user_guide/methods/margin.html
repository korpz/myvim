Margin

This method lets you easily add some margin to form elements. It is especially useful if used right after the last accessed element.

margin (integer margin, [string position])

  • margin: the margin in pixel
  • position: optional, can be "left", "right", "top" or "bottom"; default is "left"

This method usage adds a margin for the last accessed element.

margin (string name, integer margin, [string position])

  • name: the element name
  • margin: the margin in pixel
  • position: optional, can be "left", "right", "top" or "bottom"; default is "left"

This method usage adds a margin for any element of the form.

Usage example

$this->form
->submit('Submit')
->reset('Reset')
->margin(10); // adds 'margin-left: 10px' to the reset element
or 
$this->form
->submit('Submit')
->margin(10, 'right') // adds 'margin-right: 10px' to the submit element
->reset('Reset');
