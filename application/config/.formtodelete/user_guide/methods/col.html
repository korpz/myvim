Columns Method

This will let you create columns in your forms. You can use this to show two fields next to each other, for example zip-code and city name.

col (integer width, [string float, string text-align])

  • width: width of the column, 0 for resetting columns, leave empty for auto-width
  • float: where to float the column. Can be "left", "right" or "none"
  • text-align: can be "left", "right", "center" or "justify"

Usage example

Create a column 150 px wide, floating left and align the text to the right:

$this->form->col(150, 'left', 'right');

Create a second column with auto-width, filling the rest of the page to the right:

$this->form->col();

Close the column going back to normal:

$this->form->col(0);

City/zip on one line

// cols help you to easily create floating elements
$this->form->col(150, 'left', 'right')

// since the 'zip' label cannot be placed in the default label position
// we have to create individual labels in front of them
->label('City', 'city', 'class=left combine')
->label('/Zip', 'zip', 'class=left combine, style=padding-right:10px')

// this col creates a column with an automatic width
->col()

// now we will create both the 'city' and the 'zip' element
->text('city|city', '', '', '', 'style=width:108px,element_suffix=""')
		
// let's add the 'zip' field and use a custom validation function
->text('zip|zip', '', 'exact_length[5]|is_natural|callback_valid_zip', '', 'style=width:60px')

// and add some margin-left to the last_accessed element ('zip')
->margin(10)		

// and close the the columns
->col(0);
