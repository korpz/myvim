Label Display

In order to control how labels will be displayed you need to define some values in the configuration file:

$config['defaults'] = array(
   'label' => array(                  // this will be applied to ALL labels
      'position' => 'before',
      'class' => 'left',
      'required_class' => 'req',
      'required_flag' => '*'
   ),
   'checkbox' => array(
      'label' => array(               // these settings will overwrite general settings above for all checkbox labels
            'position' => 'after',
            'class' => 'right'
      )
   ),
);

position

The label tag can be inserted either before or after the element tag.

class

If set, this class will be added to the <label> tag. This is very useful to apply the correct CSS style for labels in different positions.

required_class

If set, this class will be added to the <label> tag for all elements (except labels of checkboxes and radiobuttons) that are required by the validation rules. For example you can use this to style all required labels in bold font.

required_flag

If set, this code will be added after the label text in all elements (except labels of checkboxes and radiobuttons) that are required by the validation rules. For example you can use this to add an asterisk to all labels of required fields.

Of course you can also supply additional valid attributes (like 'style' etc.) to labels.
