Attributes

For most form elements you can supply attributes that will be applied to the generated form tag via the attributes parameter in the form element method.

Element Attributes

To include attributes for the element, simply supply an attribute string or an associative array with attributes:
$this->form->text('name', 'Label', 'trim|xss_clean', '', 'class=for_element');

// or

$attributes = array('class' => 'for_element');
$this->form->text('name', 'Label', 'trim|xss_clean', '', $attributes); will generate <label for="name">Label</label> <input type="text" name="name" class="for_element" />

Label Attributes

To include attributes for the element's label, supply the attributes enclosed in an array like this: $this->form->text('name', 'Label', 'trim|xss_clean', '', array('class=for_label'));

// or

$attributes = array('class' => 'for_label');
$this->form->text('name', 'Label', 'trim|xss_clean', '', array($attributes)); will generate <label for="name" class="for_label">Label</label> <input type="text" name="name" />

Element and Label Attributes

To include attributes for both the element and the label, supply the attributes like this: $this->form->text('name', 'Label', 'trim|xss_clean', '', array('class=for_element', 'class=for_label'));

// or

$element_attributes = array('class' => 'for_element');
$label_attributes = array('class' => 'for_label');
$this->form->text('name', 'Label', 'trim|xss_clean', '', array($element_attributes, $label_attributes)); will generate <label for="name" class="for_label">Label</label> <input type="text" name="name" class="for_element" />
