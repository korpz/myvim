Creating a Form

After successful installation creating a form with the form generation library is as easy as this:

1. Load the library

$this->load->library('form');

2. Add elements to the form

$this->form
->text('Your Name', 'name', 'required')

// add more elements here
->submit();

3. Get the form output (and validate form data)

Calling the get() method will also validate the form data.

$data['form'] = $this->form->get();

4. Do stuff subject to validation

if ($this->form->valid)
{
    // validation passed
    redirect('success');
}
else
{
    $data['errors'] = $this->form->errors;
}

5. Get validated post data

If the form has been submit and passed validation you can easily retrieve the post data via the get_post() method.

if ($this->form->valid)
{
    $post_vars = $this->form->get_post();
    echo $post_vars['name'];
}
