Chaining

To make form creation even simpler and more efficient you can use method chaining to build your forms.

So instead of writing this: $this->load->library('form');

$this->form->open('login');
$this->form->text('username', 'Your Username', 'trim|alpha_numeric|max_length[30]|xss_clean');
$this->form->pass('password', 'Your Password', 'trim|alpha_numeric|max_length[20]|xss_clean');
$this->form->indent(200);
$this->form->checkbox('loggedin', 'yes', 'I want to stay logged-in');
$this->form->submit();
$this->form->reset();
$this->form->onsuccess('redirect', 'login/success');

echo $form->get(); you can make it shorter by chaining all methods after another like this: $this->load->library('form');

$this->form
->open('login')
->text('username', 'Your Username', 'trim|alpha_numeric|max_length[30]|xss_clean')
->pass('password', 'Your Password', 'trim|alpha_numeric|max_length[20]|xss_clean')
->indent(200)
->checkbox('loggedin', 'yes', 'I want to stay logged-in')
->submit()
->reset()
->onsuccess('redirect', 'login/success');

echo $form->get();

Note: Please note that method chaining only works with PHP 5
